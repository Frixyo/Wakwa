Commande de base pour Haulstein

1. Install dependencies



   ```bash
      E:

      cd repo

      cd Wakwa

      npm install
      npm i
   ```

2. Start the app

   ```bash
      npm run start
   ```

## Reinstaller la BD

   go into assets/db
   delete mydatabase.db
   ```bash
      sqlite3 mydatabase.db < init.sql
   ```
3. Expo go

   CTRL+C pour fermer l'appli
